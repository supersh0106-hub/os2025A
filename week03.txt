Git branch -M main
	브랜치 이름을 main으로 변경함 (윤리적 문제 방지)
Git remote -v
	현재 설정된 원격 리포지토리 주소 보기
Git remote add origin https://깃허브리포지토리주소
	위 주소를 원경 리포지토리로 설정
Git push -u origin main
